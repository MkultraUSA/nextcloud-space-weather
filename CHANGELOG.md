# Changelog

## 1.0.0 — 2026-05-26

### Initial Release
- Real-time KP Index monitoring with geomagnetic storm status
- Solar Flux (F10.7) measurements in solar flux units (sfu)
- X-Ray Flux monitoring with NOAA alert level classification
- Aurora Forecast imagery from NOAA SWPC
- HF Band Propagation conditions via HamQSL (80m–2m)
- D-RAP Absorption prediction maps
- SDO Solar Imagery gallery (AIA 94Å, 193Å, 211Å, HMI Magnetogram, Continuum)
- Weather Satellite Imagery (GOES-16, GOES-17)
- Image proxy controller for CSP-safe external image loading
- TTL-based caching (5min real-time, 30min forecast, 60min daily)
- Manual refresh with cache invalidation
- Theme-neutral responsive CSS (supports light and dark Nextcloud themes)
- CSP-safe vanilla JavaScript (no build step, no frameworks)
- Settings page with data source info, app details, and feature request form
- Admin settings panel with configurable cache TTL, API timeout, and data source toggles
- Credits and Attribution section acknowledging NOAA, NASA, HamQSL, and GOES
- Nextcloud 27–35 compatibility
- AGPL-3.0-or-later license
